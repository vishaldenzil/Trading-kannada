# Trading Kannada Website

A Next.js client website for Trading Kannada, built from the public YouTube channel and client profile.

## Live Site

https://i-want-to-create-website-for.vercel.app

## Files

- `app/page.tsx` - page structure and content
- `app/layout.tsx` - metadata and global layout
- `app/components/Header.tsx` - responsive navigation
- `styles.css` - responsive visual design

## Local Development

```bash
npm install
npm run dev
```

## Deployment

This site is deployed on Vercel.
